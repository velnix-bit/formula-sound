Formula Sound 公開準備版

1. index.html をブラウザで開いて動作確認してください。
2. about.html の運営者名を記入してください。
3. contact.html に専用メールアドレスまたはお問い合わせフォームを追加してください。
4. 独自ドメイン決定後、sitemap.xml と robots.txt の example.com を置き換えてください。
5. 広告・アクセス解析を導入する際は privacy.html を実態に合わせて更新してください。
6. GitHubへフォルダ内の全ファイルをアップロードし、Cloudflare Pages等で公開できます。
